## SWC Admin

A ready-to-use backend template that integrates commonly used modules.

## Features

* Base on Vite, React, Typescript, SWC
* Conventional Routing with vite-plugin-pages
* Base Layout with Ant Design Pro

## Qiuck Start

```bash
$ pnpm i
$ pnpm run dev
```
