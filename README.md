## SWC Admin
一个开箱即用的后台模版，集成常用模块。

## 特征
- 基于vite swc，超快打包速度、热更新速度。
- 约定式路由，快速定位bug。
- 基于 Pro components 开发，开发业务代码也很快

## 快速开始

```bash
# 安装依赖
$ pnpm i
# 启动服务
$ pnpm run dev
```
